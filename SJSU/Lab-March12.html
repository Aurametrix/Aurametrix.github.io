2. Log onto the Biology WorkBench, select Session Tools and create a new session

Select "Nucleic Tools". Scroll down in the menu to "Add New Sequence".

Paste fasta file (or download it) and name each species accordingly. Save.

Select all the sequences or pick those you wish to align by clicking the box next to each file, and then select CLUSTALW. Run

Select Import Alignment(s). This will save this alignment in a single file for further analysis.

Submit (with Default parameters), Now change to rooted tree

Create a phylogenetic tree by DRAWTREE or DRAWGRAM.

DRAWTREE draws a rooted tree.

DRAWGRAM draws an unrooted tree.

 

Examine your tree.

Which species appear to be the most closely related?

From your tree, does it appear that Neanderthals were direct ancestors of humans? Where is the branching point from other apes?
